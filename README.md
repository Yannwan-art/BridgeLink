🔗 BridgeLink — GERT Network Simulation & Decision Support System
**BridgeLink** is an interactive web-based system for visual modeling and simulation of **GERT (Graphical Evaluation and Review Technique)** stochastic networks. It is designed to support decision-making in supply chain management, project planning, and logistics under uncertainty.

🌐 **Live Demo:**https://yannwan-art.github.io/BridgeLink/

🎯 Overview
BridgeLink provides a visual, browser-based environment for building and analyzing GERT network models. Unlike traditional analytical tools, it allows users to:
- **Visually construct** network models with nodes and edges
- **Drag and bend edges** to improve diagram readability
- **Run Monte Carlo simulations** to estimate process completion time
- **Compare multiple scenarios** with different probability and time parameters
- **View results** in real-time through tables and bar charts
The system is built as a lightweight web application with no installation required — simply open the link in any modern browser.


🚀 Key Features
| Feature | Description |
| Interactive GERT Modeling | Build and edit stochastic networks with branches, parallel paths, and feedback loops |
| Drag-to-Bend Edges | Improve diagram clarity by dragging edge labels to curve lines |
| Monte Carlo Simulation | Run N simulations to compute mean, standard deviation, and coefficient of variation |
| Multi-Scenario Comparison | Switch between parameter schemes (S0, S1-1, S2-1, etc.) and compare results |
| Real-time Visualization | View network graphs with G6 and results with ECharts |
| Cross-Platform | Works on Windows, Linux, macOS, and mobile devices via web browser |
| Zero Installation | No software to install — runs directly in the browser |

🎮 How to Use
1. **Open the application** — [https://yannwan-art.github.io/BridgeLink/](https://yannwan-art.github.io/BridgeLink/)
2. **Explore the network** — Click nodes/edges to view parameters
3. **Bend edges** — Drag labels to curve lines for better visibility
4. **Run simulation** — Set N (iterations) and choose a parameter scheme (S0, S1-1, etc.)
5. **View results** — Statistics update automatically; click "Generate Chart" for visual comparison
6. **Compare scenarios** — Switch schemes and re-run to compare outcomes
